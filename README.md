# PKU Planner - 智能课程与 DDL 管理专家

PKU Planner 是一款专为北京大学学生打造的现代化学习管理工具。它不仅是一个课程表，更是一个智能化的 DDL 追踪中心，通过视觉化提醒与 AI 技术帮助你高效掌控学期进度。

## ✨ 核心亮点

### 🌈 视觉驱动的任务紧急度提醒
不再需要逐个翻找作业日期！PKU Planner 创新性地将任务状态与课程表色彩深度联动。课程格子会根据最近一个任务的剩余时间自动变换颜色：
- 🔴 **红色**：任务已逾期，需立刻处理。
- 🟠 **橙色**：任务就在今晚截止，进入紧急状态。
- 🟡 **黄色**：DDL 正在逼近（3天内）。
- 🔵 **蓝色**：近期有 DDL 但时间尚宽裕。
- 🟢 **绿色**：暂无任务待办，可以安心享受学习或休息。

### 🤖 AI 智能课表导入
接入 Gemini 与 豆包 (Doubao) 双大模型。
- **拍图即录**：只需上传一张课表截图，AI 会自动识别课程名称、教室、教师及节次。
- **智能纠错**：自动合并重复课程，智能解析单双周规则。
- **多格式支持**：除了图片识别，也支持经典的 CSV 和 JSON 格式。

### 💡 智能“今日建议”系统
Dashboard 页面左侧会根据你的学业现状动态生成建议：
- **逾期预警**：⚠ 提醒处理已超期的顽固任务。
- **高负荷预警**：📚 识别待办任务过多的课程。
- **进度激励**：🎉 实时计算任务完成率，见证你的每一次进步。
- **实时同步**：修改、新增、清空数据，建议即刻刷新，零延迟反馈。

### 🖱️ 非侵入式悬浮预览
无需点击进入详情页，只需将鼠标悬停在课程表格子上，即可通过 **DDL 预览弹窗** 快速查看该课程的所有待办事项，极大提升信息获取效率。

## 📅 功能介绍

### 课程管理
- 适配北大教务系统逻辑，支持单周/双周/全周设置。
- 精确到教室与教师的详细信息管理。
- 实时显示的学期周数进度条，帮你找准学期坐标。

### 任务与 DDL 管理
- 课程关联任务，支持多级优先级设定。
- 智能倒计时提示，精确显示“还有多少天”。
- 快捷任务创建：在课程格子上直接右键即可为该课添加作业。

### 数据统计与健康
- 实时统计今日任务数、本周 DDL 数。
- 可视化学期进度条，量化你的学期旅程。

## 🚀 快速上手

1. **导入课表**：
   - 进入设置 -> 点击「📥 导入课表」。
   - 推荐尝试「图片导入」，通过 AI 快速解析你的教务系统课表截图。
2. **添加任务**：
   - 在首页双击空白格子添加课程，或在已有课程上右键选择「添加任务」。
3. **查看建议**：
   - 关注首页左侧的「今日建议」，根据颜色提醒合理规划学习时间。
