# PKU Planner 使用说明

## 简介

PKU Planner 是为北京大学学生设计的课程与任务管理工具，帮助你更好地规划学习生活。

## 功能介绍

### 📅 课程管理
- 在首页添加和管理你的课程
- 支持单周/双周课程
- 查看课程详情、添加笔记和资料

### 📝 任务管理
- 为每门课程添加作业和考试任务
- 设置截止日期和优先级
- 追踪任务完成进度

### ⏰ DDL 提醒
- 开启提醒功能，避免忘记作业截止时间
- 可自定义提醒频率（15分钟/30分钟/1小时/2小时）

### 📊 数据统计
- 查看本周/本学期的任务完成情况
- 了解学习进度和效率

## 导入课表

### 支持格式

#### 1. 文本格式 (.txt / .csv)
每行格式：`课程名, 教师, 时间, 地点, 周类型`

示例：
```
计算机系统导论, 陈晓红, 周二 3-4节, 理科1号楼101, 单周
数据结构与算法, 郭炜, 周四 7-8节, 理科1号楼203, 全周
```

时间格式说明：
- `周一 1-2节` 表示周一第1-2节
- 支持格式：`周X 第A-B节` 或 `星期X A-B节`

周类型：
- `单周` - 单周上课
- `双周` - 双周上课
- `全周` - 每周都上课（默认）

#### 2. JSON 格式 (.json)
```json
{
  "courses": [
    {
      "name": "计算机系统导论",
      "teacher": "陈晓红",
      "location": "理科1号楼101",
      "day": 2,
      "startPeriod": 3,
      "endPeriod": 4,
      "weekType": 1
    }
  ]
}
```

字段说明：
- `day`: 1-7（周一到周日）
- `startPeriod`: 起始节次
- `endPeriod`: 结束节次
- `weekType`: 0=全周, 1=单周, 2=双周

### 导入步骤
1. 打开设置页面（侧边栏齿轮图标）
2. 点击「📥 导入课表」按钮
3. 选择你的课表文件
4. 确认导入成功

## 数据管理

### 导出数据
支持将所有课程和任务导出为 CSV 格式，方便备份或在其他软件中使用。

### 备份数据
点击「💿 备份数据」可以将当前数据备份到指定位置。

### 重置
- **重置引导**：重新显示欢迎引导
- **恢复出厂设置**：清除所有数据并重置应用

## 学期设置

在设置页面可以修改学期起止日期，系统会自动计算当前周数和学期进度。

## 快捷操作

- 点击首页「+」添加课程或任务
- 长按任务可编辑或删除
- 课程卡片显示本周课程概览
